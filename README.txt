Pet Family - Projeto Estático

Estrutura:
- index.html
- css/styles.css
- js/script.js
- assets/images/*
- assets/icons/*
- assets/fonts/*

Como executar:
1. Abra o arquivo index.html no navegador.
2. Opcional: rode um servidor estático local (ex.: python -m http.server) dentro da pasta pet-family.

Checklist rápido:
- Menu mobile abre/fecha corretamente
- Tema claro/escuro persiste no recarregamento
- Âncoras fazem rolagem suave
- Formulário valida campos e mostra mensagem de sucesso
- Layout responsivo em 900px, 600px e 400px

Observação:
- Um asset remoto original (imagem de acessórios) não estava mais disponível no host de origem e foi substituído por uma imagem local de fallback para manter a apresentação offline completa.
